## üåü Key Features

### 1. The Launchpad (Hero Section)
A massive, screen-filling typographic experience with a real-time generated **3D Starfield**. The text parallax effects give a sense of depth as you scroll into the void.

### 2. Destination Trajectories
Interactive, **Holographic Cards** for:
*   **Artemis Base (Moon)**: Low-gravity luxury.
*   **Olympus Mons (Mars)**: Terraformed volcanic resort.
*   **Orbital Lounge (Saturn)**: Zero-G fine dining in the rings.

Each card features a tilt-parallax effect and "Active Camo" image reveals.

### 3. The Fleet (Digital Hangar)
A horizontal scrolling "hangar" showcasing our flagship vessels:
*   **Aetheris-X**: Deep Space Cruiser.
*   **Void Runner**: Stealth Orbital Shuttle.
*   **Star Hopper**: Atmospheric Glider.

Features **scanning laser animations**, tech-grid backgrounds, and responsive ship specifications.
